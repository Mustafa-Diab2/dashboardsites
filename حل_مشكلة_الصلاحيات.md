### الخطوة 2: إنشاء المستند
1. اضغط على "Start collection" أو اذهب لمجموعة `users`

### الخطوة 3: إضافة الحقول
أضف هذه الحقول بالضبط:

| اسم الحقل | النوع | القيمة |
|-----------|-------|--------|
| `id` | string | `PEVTqUVMPzSTmTETWXi1Bzkvx5D2` |
| `email` | string | `nourkhaleds1223@gmail.com` |
| `fullName` | string | `Nour Khaled` |
| `role` | string | `frontend` |
| `createdAt` | timestamp | (اضغط "timestamp" واستخدم الوقت الحالي) |

### الخطوة 4: حفظ
اضغط على زر "Save".

### الخطوة 5: إعادة تحميل المتصفح
1. ارجع لتطبيقك
2. حدث الصفحة (F5 أو Ctrl+R)
3. أو سجل الخروج ثم سجل الدخول مرة أخرى
4. يجب أن تستطيع الآن الدخول للتطبيق بصلاحيات مستخدم عادي!
