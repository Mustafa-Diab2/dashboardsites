# **App Name**: TaskWise Reports

## Core Features:

- CSV Export: Export team task reports as CSV files for further analysis and sharing.
- PDF Export: Export team task reports as PDF documents for easy distribution and archiving.
- Task Summary by Member: Aggregate the information about how many tasks have been assigned to each user, how many of them are in the 'backlog' state, 'in progress' state, under review or in 'done' state
- Generate Insights and Recommendations: AI tool which reviews the tasks summary by member to detect workload imbalances or inefficiencies. Provides actionable recommendations for task redistribution or process improvements.
- Charting: Use the recharts library to show information about project state as bar charts and pie charts.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) to convey a sense of calm and reliability.
- Background color: Very light blue (#F0F8FF) to provide a clean and unobtrusive backdrop.
- Accent color: Muted violet (#A094C7) for interactive elements and highlights, providing a touch of sophistication.
- Headline font: 'Space Grotesk' sans-serif, known for a modern aesthetic, paired with 'Inter' for body text.
- Use clean, minimalist icons from Lucide to represent different export formats and data visualizations.
- Employ a grid-based layout to organize reports and charts, ensuring a balanced and responsive design.
- Subtle transitions and animations when loading data or switching between reports to enhance user engagement.
